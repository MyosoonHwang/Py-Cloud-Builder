# ☁️ NHN Cloud Network Auto-Builder

**NHN Cloud (OpenStack)** 환경에서 VPC와 서브넷을 쉽고 안전하게 관리하는 파이썬 자동화 도구입니다.
복잡한 콘솔 작업 없이, 터미널에서 **리소스를 시각적으로 조회**하고 **중복 걱정 없이 생성**할 수 있습니다.

## ✨ 주요 기능 (Key Features)

* **🔐 하이브리드 인증 (Smart Auth):**
    * 아이디/비밀번호만 입력하면 **참여 중인 프로젝트(Tenant)를 자동으로 검색**해 연결합니다.
    * 자동 조회가 실패할 경우, Tenant ID 수동 입력 모드로 자동 전환되어 끊김 없이 작업할 수 있습니다.
* **🛡️ 3중 스마트 유효성 검사:**
    1. **CIDR 중복 방지:** 기존 VPC 대역과 겹치는지 계산하여 경고하고, 사용 가능한 대역을 추천합니다.
    2. **이름(Name) 중복 방지:** 이미 존재하는 VPC나 서브넷 이름을 사용할 수 없도록 막습니다.
    3. **범위(Scope) 검사:** 서브넷이 VPC의 CIDR 범위를 벗어나지 않는지 검증합니다.
* **👀 스마트 필터링 (Smart Filtering):**
    * NHN Cloud의 공용 네트워크(`Public Network`)나 다른 프로젝트의 리소스는 제외합니다.
    * 오직 **내가 현재 프로젝트에서 생성한 리소스**만 깔끔하게 필터링하여 보여줍니다.
* **📊 트리 뷰 시각화:** VPC와 그 안에 속한 서브넷 구조를 계층형 트리(Tree) 형태로 직관적으로 출력합니다.
* **⚡ .env 환경 변수 지원:** 매번 로그인할 필요 없이 `.env` 파일에 계정 정보를 저장하여 **원터치 로그인**이 가능합니다.

# ☁️ NHN Cloud Network Manager

NHN Cloud(OpenStack 기반)의 VPC 및 서브넷 리소스를 쉽고 안전하게 관리하기 위한 파이썬 자동화 도구입니다.
복잡한 콘솔 작업 없이, CLI 환경에서 **리소스를 조회**하고 **중복 없이 생성**할 수 있습니다.

## ✨ 주요 기능

* **🔐 하이브리드 인증:** 아이디/비밀번호만 입력하면 **참여 중인 프로젝트(Tenant ID)를 자동으로 검색**합니다.
* **🛡️ 스마트 중복 방지:**
    * **대역(CIDR) 충돌 방지:** 기존 VPC 및 공용 네트워크와 겹치는지 자동으로 계산합니다.
    * **이름(Name) 중복 방지:** 이미 존재하는 리소스 이름은 사용할 수 없도록 막습니다.
* **📊 리소스 시각화:** 현재 프로젝트에 속한 VPC와 서브넷 구조를 **트리(Tree) 형태**로 직관적으로 보여줍니다.
* **👀 스마트 필터링:** 공용 네트워크(Public Network)나 다른 프로젝트의 리소스는 제외하고, **내 프로젝트의 리소스만** 깔끔하게 보여줍니다.
* **⚡ .env 지원:** 매번 로그인할 필요 없이 환경 변수 파일로 자동 로그인이 가능합니다.
